Animatub, YouTube a gif

La última vez hice un script para pasar video a gif animado, ahora es YouTube a gif animado, Animatub.

Para acortar las cosas, últimamente había estado ocupando el antecesor Anima para pasar video a gif animado. Primero tenía que bajar el video, anotar el nombre en la terminal, y luego pasar lo que quería a gif.

Ahora decidí acortar el proceso y tomar el video directo desde YouTube y que pase lo seleccionado a gif animado. Todo esto con Animatub.

Descarga Animatub

Puedes descargar el archivo animatub.sh desde GitHub (el zip).

Eso si, esto requiere de FFmpeg, youtube-dl, y convert (viene incluido con imagemagick) para funcionar. Los requerimientos los puedes descargar desde APT en elementary OS/Ubuntu, o Homebrew en OS X.

Modo de uso

Es siempre el mismo orden:

./animatub.sh https://youtu.be/qwerty 00:01:00 5

Según el ejemplo de arriba, desde el minuto 1 tomará 5 segundos para crear el gif animado. El gif se guardará con el mismo nombre del video.

Los gifs se guardan con una resolución de ancho de 480px, respetando la relación de aspecto de la imagen. Usa tu navegador web para ver los gifs animados.
