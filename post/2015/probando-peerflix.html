Probando Peerflix

Estuve probando Peerflix, una app de terminal para ver video en streaming de torrents.

Demasiado bueno para ser cierto

Gracias al motor torrent-stream, Peerflix puede existir. Solo toma el torrent o enlace magnético y comienza a cargar desde el principio del video. Se puede utilizar con múltiples reproductores (mpv, MPlayer, VLC), y agregar subtitulos.

Instalar

Si estás usando Ubuntu 14.04 o elementary OS Freya, primero agrega este repositorio, y instala nodejs (viene incluido npm). En OS X instala npm usando Homebrew. Instalar peerflix es igual para todos:

sudo npm install -g peerflix

Usar

Ahora para reproducir contenido se usa así, junto con mi reproductor favorito mpv (otros: --mplayer --vlc):

peerflix archivo.torrent --mpv peerflix "magnet:enlace" --mpv

Para agregar subtitulo y que el reproductor lo lea (lista de sitios de torrents y subtitulos):

peerflix archivo.torrent --mpv --subtitles subtitulo.srt

Dependiendo de la velocidad de su conexión, quizás tengan que pausar al inicio un minuto el reproductor, para que no haya tropiezos más tarde. Si quieren saber más sobre como usar Peerflix: peerflix --help
