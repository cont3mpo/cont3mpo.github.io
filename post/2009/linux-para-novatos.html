Linux para novatos

Daré una idea de qué involucra usar Linux.

Existen diferentes Sistemas Operativos que utilizan núcleo Linux, debido a que Linux es un núcleo libre y puede utilizarse para todo tipo de cosas, ejemplo: sistemas operativos como Android o Ubuntu.

Sistemas Operativos de escritorio

Un OS de escritorio completo, el cual trae todo tipo de aplicaciones, puede ser Ubuntu, Fedora, OpenSUSE, Debian, Mandriva, etc.

Los OS de escritorios "basados en X distribución" son muchos, por eso siempre hay que ir a los más usados por el amplio soporte.

Gracias a GRUB o BURG, puedes seleccionar varios sistemas operativos que tengas instalados. Ya que al prender la máquina, aparecerá un menú con los OS.

Repositorios

Cada OS de escritorio que utiliza Linux cuenta con sus propios repositorios o centros de aplicaciones. ¿Qué es eso? Es como un indice para todo el software disponible para su OS. Esto quiere decir que para instalar y mantener todos los días actualizadas tus aplicaciones, ni siquiera necesitas abrir un navegador Web. Y lo mejor de todo es que puedes añadir repositorios de terceros, o sea de sitios como Launchpad, que es donde dan cabida a los repositorios personales, y brindan soporte para todas las aplicaciones que son hospedadas ahí.

Escritorios

En los OS de escritorio hay varios entornos de escritorios, existe KDE, Xfce, GNOME, Fluxbox, LXDE y Unity. Esos serían los entornos de escritorio más populares.

Aplicaciones

Reproductores de música y administrar tu música en un dispositivo como el iPod, puedes probar con Banshee, Songbird, Exaile, Rhythmbox, Amarok, etc.

Reproducir videos existe VLC, SMPlayer, GNOME Mplayer, Totem, y Xine. Esos serían los más importantes, aunque hay varios más, yo ocupo GNOME MPlayer para ver todos mis videos.

Editores de video como Avidemux, PiTiVi, Kino, Cinelerra (uso profesional), Lives, Kdenlive, WinFF (utiliza FFmpeg), o OpenShot Video Editor. Y para audio tenemos al poderoso Audacity. En este enlace hay más editores de video para Linux.

Editores de video CLI existen MEncoder y FFmpeg.

Ripear o encodear DVD's se puede utilizar AcidRip, dvd::rip, OGMRip, y Thoggen DVD Ripper (MEncoder y FFmpeg también sirven para encodear por linea de comandos). Para creación de DVD's pueden utilizar CD/DVD Creator, ManDVD, DVD Styler, QDVDAuthor, y DeVeDe.

Grabar discos con datos o imágenes ISO existe Brasero, K3b, GnomeBaker, Nero Linux (de prueba), o Etcher.

Editores de imágenes existe Gimp, Inkscape, Xara Xtreme, Pinta, y Krita (forma parte de KOffice). También mencionar a Blender, que es una gran herramienta para crear figuras tridimensionales.

Suite Ofimáticas como LibreOffice, OpenOffice, KOffice. También existe Gnome Offfice, y cuyas aplicaciones son Abiword (un procesador de textos liviano), Gnumeric (plantillas tipo excel muy liviano). Y recordar que siempre está disponible Google Docs, ahí respaldo todo lo que crea que pueda utilizar en otro computador.

Navegadores Web hay por montones, Firefox es muy potente al igual que Google Chrome, también se encuentra Opera, Arora, Midori, IceWeasel, Konqueror, etc.

Descargar torrents existe Transmission, Vuze (Azureus), KTorrent, hasta el mismo uTorrent (gracias al emulador Wine), o WebTorrent Desktop.

Mensajería instantanea existe Pidgin, Empathy, Emesene, Mercury, Kopete, aMSN, WhatsApp Web.

Editores de código puedes utilizar Atom, Sublime Text, Eclipse, NetBeans, CodeLite, MonoDevelop, etc.

Crear sitios Web gráficamente, pues utiliza BlueFish, Geany, KompoZer (Nvu), Quanta Plus, Screem.

Esos son ejemplos de los programas más habituales, siempre pueden ir a su gestor de aplicaciones, como Ubuntu Software Center (Centro de aplicaciones de Ubuntu) y buscar por palabras o categorías para encontrar alguna otra aplicación.

Además no está demás ver esta lista de 100 aplicaciones de uso cotidiano en Linux.

Emuladores

Si quieren probar aplicaciones de Windows en un OS Linux, para eso existe Wine, que se dedica a la emulación de aplicaciones Windows en vivo y directo, aunque siempre son los más populares los que funcionan, ojo, no todos las aplicaciones de Windows funcionan con Wine, solo los que se encuentran disponibles en su base de datos de aplicaciones publicada en su sitio. Aunque sirve de mucho mejor utilizar el buscador de aplicaciones de Wine.

VirtualBox es un virtualizador de sistemas operativos, ahí pueden probar el sistema operativo que quieras cuando te aburras, como Mac OS X, Windows, BSD, o otras distros de Linux.

VMware Worstation o VMware Player (para Linux), y también es un virtualizador de Sistemas Operativos. El único inconveniente es que cuenta con un periodo de prueba limitado, ya que esta es una aplicación de pago. Puedes emular varios sistemas operativos.

QEMU es otro conocido emulador. Puedes instalar y probar un OS directamente como si fuese una ventana más.

¿Y la Terminal?

Ahora existen muchas distribuciones que solamente utilizan Interfaz Gráfica de escritorio para mantener tu sistema optimo, y solo los usuarios avanzados hacen uso de la Terminal de comandos. Aunque nunca está demás saber uno que otro comando para alguna emergencia.
