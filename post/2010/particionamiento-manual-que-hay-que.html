Particiones: Lo que hay que saber

Cuando se instala una distro de Linux popular y fácil de usar como los son Ubuntu, Fedora, Mandriva, Debian, o openSUSE, el instalador realiza un formateo del disco duro completo, o sea... borra todo. Para luego empezar a instalar el sistema operativo desde cero, y además crea una pequeña swap (la swap es como una memoria RAM extra de intercambio). Y eso sería, es super simple lo que hace una distro Linux fácil de usar, todo lo hace automático, solo hay que aceptar todo lo que salga en la instalación gráficamente.

La gracia viene cuando tenemos particionado el disco duro. Las particiones se refieren a cuando se divide el disco duro en las partes que uno desee para instalar otros sistemas operativos (OS) en el mismo disco, o también para dejar un espacio aparte para guardar lo que sea (como por ejemplo; si le pasa algo al sistema operativo y hay que volver a formatear, pero si tenemos datos guardados en una partición aparte no les ocurriría nada

Sistema de Archivos y Directorios

El sistema de archivos es lo que se utiliza al momento de formatear e instalar el sistema operativo. Como por ejemplo: En Windows se utiliza el sistema de archivos NTFS o FAT32. En Mac OS X se utiliza ZFS. Y en Linux se puede utilizar ext3, ext4, Btrfs, o XFS.

Ahora se masificó el uso de ext4, y usaremos ese (aunque Btrfs es igual de bueno).

Cuando se particiona manualmente, hay solo dos cosas que se tienen que hacer obligatoriamente. Y esas son: Crear el directorio raíz: " / ", y crear la swap. La swap no utiliza directorio, es solo un pequeño particionamiento de intercambio de memoria. El directorio raíz es solo una barra invertida: " / ", y ahí es donde se instala todo el sistema y los sub-directorios que parten desde " / ", por eso es obligatorio crearlo. La partición en Linux se hace gráficamente desde cualquier disco de instalación de las distros mencionadas arriba.

Agregando más directorios

Si ya hay otros sistemas operativos instalados en el disco, o si hay otras particiones para guardar datos. Se pueden agregar directorios para tenerlos ordenados y/o darles permisos de solo lectura a cada directorio mas adelante cuando ya esté todo instalado (hace un tiempo escribí sobre los permisos en los directorios para evitar a los intrusos). La creación de estos directorios es absolutamente opcional, si quieren los crean, pero servirá para que el sistema los tome en cuenta en cada partida del sistema. Y ojo: no se formatean si ya hay un sistema operativo (OS) instalado en ellos.

¿Que se puede agregar?

A las particiones aparte se les puede colocar el nombre que uno quiera, como un ejemplo...

Si tienen Windows, pueden crear el directorio: /win
Si tiene Mac OS X, pueden crear el directorio: /osx
O si tienen una partición en que solo guardan datos: /data o /cosas

Todos estos son directorios que aparecerán en su directorio raíz " / " una vez instalado el sistema. No se formatean (a menos que no haya nada ahí), si formatean se borrará lo que hay en esas particiones.

El directorio de usuarios

El directorio de usuarios lleva por nombre "/home". Ahí es donde se guardan las configuraciones de cada usuario creado, y se guarda absolutamente todo lo relacionado con el escritorio, archivos de configuración de las aplicaciones y todo tipo de configuraciones del usuario.

Este directorio también es opcional crearlo en el particionamiento manual, ya que "/home" se crea automáticamente cuando se instala el sistema. Esto es solo para tenerlo en una partición aparte y mantener guardadas para siempre las configuraciones del usuario, y no perder todo eso en un próximo formateo. Se puede formatear en ext4, al igual que el sistema operativo que está instalado en " / ". Si el directorio "/home" ya existe con anterioridad, no lo formateen.

Particiones, directorio y formateo

Ahora pasamos rápidamente a como se vería todo, y lo que hay que hacer en cada caso:

Obligatorios

  • Directorio raíz: / (utilicen ext4 para formatear)
  • La swap (memoria de intercambio): No utiliza directorio, dejen unos 256 MB de memoria.

Opcionales

  • Directorio de usuarios: /home (utilicen ext4 para formatear, si ya existe no formatear)
  • Directorio de Windows: /win (si ya hay un OS instalado no formatear)
  • Directorio de Mac OS X: /osx (si ya hay un OS instalado no formatear)
  • Directorio de otros datos: /data (si ya existe no formatear)

Y eso. No es una gran ciencia, es fácil si se recuerda lo que hay que hacer.
