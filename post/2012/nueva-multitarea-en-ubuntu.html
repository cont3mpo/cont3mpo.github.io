El nuevo Multitarea de Ubuntu

El equipo de Diseño de Canonical publicó los planes de diseño e interacción para un nuevo multitarea o cambiador de tareas en Unity. Y que partirá desde Ubuntu 12.10 en adelante, esto involucra primeramente al gestor Compiz que utiliza Unity. Algunas de las novedades a continuación...

Interacción Multitarea

  • Para manejar varias ventanas a la vez se podrán desmaximizar en una vista multitarea gracias a un nuevo icono en el lanzador de Unity (imagen de arriba).
  • En la vista multitarea, al momento de posicionarse sobre una ventana con el cursor, esta mostrará una barra con el botón de cerrado, para facilitar el cerrado de ventanas.
  • Las ventanas se podrán mover desde la barra que aparece al pasar el cursor.
  • Se podrán crear nuevas ventanas pero solamente desde la vista multitarea de una aplicación individual (al pulsar el icono de una aplicación con varias ventanas abiertas).
  • Se podrán mover ventanas directo desde la vista multitarea hacia otros escritorios.

Interacción de Escritorios

  • El icono para cambiar escritorios no se incorporará por defecto en el lanzador de Unity, para no crear confusión al usuario con el nuevo icono multitarea.
  • Si el usuario decide habilitar el cambiador de escritorios, entonces mostrará todos los escritorios y todas las ventanas en vista multitarea (desmaximizadas).
  • En el cambiador de escritorios se podrá amplificar un escritorio para manejar las ventanas sin la necesidad de abrir totalmente ese escritorio.
